Zap Board — Manual do Usuário

Um quadro de referências que vive dentro do seu .blend. Transforme qualquer Image Editor numa tela 2D livre, coloque suas referências nela, organize, anote — e tudo é salvo junto com o arquivo. Nada é adicionado à sua cena 3D, e nada aparece no seu render.

Versão de 30 segundos

  1. Instale e abra um Image Editor.
  2. Clique no botão Zap Board no cabeçalho desse editor.
  3. Arraste arquivos de imagem do seu gerenciador de arquivos para lá, ou aperte Ctrl+Shift+I para escolhê-los numa janela.
  4. Mova as imagens à vontade. Aperte Ctrl+P para arrumar o quadro inteiro sem sobreposições.
  5. Salve. Reabra o arquivo — o quadro está exatamente como você deixou.

1. O que é

O Zap Board transforma um Image Editor num quadro onde você solta imagens de referência. Ele é por editor, então dá para ter um quadro numa área enquanto outro Image Editor continua sendo um visualizador de imagens comum.

Ele não mexe na sua cena 3D. Nada é adicionado ao outliner, nada aparece na sua viewport e nada chega ao seu render. O quadro são dados próprios, salvos dentro do .blend junto com todo o resto.

Um quadro por arquivo nesta versão.

Requisitos

Blender 4.5 LTS ou mais recente. Verificado em 4.5, 5.0 e 5.2.

2. Instalação

  1. Edit › Preferences › Get Extensions / Add-ons › menu no canto › Install from Disk…
  2. Escolha zap_board-1.0.0.zip.
  3. Ative Zap Board na lista.
  4. Abra um Image Editor e clique no botão Zap Board no cabeçalho dele. Há também um botão de quadro no cabeçalho da 3D Viewport que converte a área atual com um clique.

Aperte N dentro do quadro para abrir a barra lateral, onde todo comando também tem um botão.

3. Trazendo imagens

Quatro caminhos, e todos colocam a imagem como um card no quadro:

  • Arraste e solte arquivos de imagem direto do seu gerenciador de arquivos sobre o quadro.
  • Import Images… (Ctrl+Shift+I) para escolher arquivos numa janela.
  • Load Folder para adicionar todas as imagens de uma pasta de uma vez.
  • Colar (Ctrl+V) para jogar no quadro uma imagem que você acabou de copiar de qualquer lugar.

Imagens coladas são embutidas na hora. Uma imagem colada não tem arquivo em disco para apontar, então o Zap Board a empacota dentro do .blend no momento em que você cola. Você não precisa fazer nada, e o card sobrevive a um recarregamento.

Colar e Reveal File são só para Windows. Os dois passam a tarefa para uma ferramenta do Windows - PowerShell para a área de transferência, Explorer para o Reveal File. No macOS e no Linux eles não fazem nada e avisam. Use Import Images… ou arraste e solte em vez de colar.

4. Organizando

Mover e selecionar

  • Clique num card para selecioná-lo, arraste para movê-lo.
  • Shift+clique para somar à seleção.
  • Arraste no espaço vazio para selecionar por caixa.
  • A seleciona tudo.
  • Com vários cards selecionados, arrastar qualquer um deles move o conjunto inteiro.
  • Botão do meio para deslocar a vista, roda para o zoom, Home para caber o quadro inteiro na tela.

Dando forma ao quadro

  • Ctrl+P — Best Fit: junta tudo sem sobreposições. Se houver algo selecionado, só a seleção é arrumada.
  • Ctrl+Alt+N / A / D / R — ordenar por nome, ordem de entrada, caminho do arquivo ou aleatoriamente.
  • Ctrl+Shift+seta — alinhar a seleção pela borda de cima, de baixo, esquerda ou direita.
  • Distribute na barra lateral — espaçamento uniforme.
  • Ctrl+Alt+seta — deixar a seleção do mesmo tamanho (por altura, largura ou maior lado).

Grade e encaixe

G mostra a grade, Shift+G encaixa nela. O tamanho da grade fica em Preferences.

5. Notas e desenho

Notas

Shift+N adiciona um card de texto. Dê um duplo clique para editar, ou edite linha por linha na barra lateral — que é também onde você define o tamanho da fonte, a cor do card e o alinhamento do texto. Fit to Text redimensiona o card para o que ele contém agora.

As notas são apoiadas num datablock de texto do Blender, então aceitam várias linhas e qualquer idioma que seu sistema consiga digitar.

Desenho

D alterna o modo de desenho. Escolha cor e espessura na barra lateral e desenhe direto sobre o quadro — marcações, círculos, setas, o que sirva para anotar uma referência. Undo Line remove o último traço, Clear remove todos. Os traços também podem ser selecionados, movidos e apagados como qualquer outro item.

6. Trabalhando com os cards

  • Opacidade — arraste o controle no card, use os botões da barra lateral, ou Ctrl+Shift+Numpad + / Numpad −.
  • Espelhar e girar — horizontal, vertical e presets de 90°.
  • Recortar — ligue o modo de recorte e arraste as alças do card. É não destrutivo: o arquivo de origem fica intacto e Reset devolve a imagem inteira.
  • Lock num card para que ele não seja arrastado ou redimensionado sem querer. Group em cards para que clicar em um mova o conjunto todo.
  • Bring to Front / Send to Back (Ctrl+Shift+A / B) para cards sobrepostos.
  • Solo esconde tudo menos a seleção. F foca um card; as teclas de colchete pulam para o próximo e o anterior.
  • Escala de cinza — Alt+G para a seleção, Ctrl+Alt+G para o quadro inteiro, para avaliar valores.
  • Export as Image escreve o quadro inteiro como um arquivo só.

Os quatro atalhos de escala de cinza e de grade ficam todos no G (G, Shift+G, Alt+G, Ctrl+Alt+G). Se o quadro de repente parecer errado, é a primeira coisa a conferir.

7. Enviando um quadro para outra pessoa

O layout fica guardado no seu .blend, mas as imagens são guardadas do jeito que o Blender guarda qualquer imagem: como um caminho para um arquivo em disco. Mande o .blend sozinho e quem abrir recebe o seu layout com os cards vazios, porque os arquivos de imagem nunca saíram da sua máquina.

Você mesmo não vai perceber isso. A sua cópia dos arquivos continua aí, então o seu quadro parece perfeito de qualquer jeito. Esta é a única coisa que vale a pena guardar deste manual.

Antes de enviar, aperte Pack Images for Sharing na barra lateral. Isso embute as imagens dentro do .blend, e aí o quadro chega completo. O preço é o tamanho: empacotar um conjunto grande de referências deixa o .blend mais ou menos do tamanho das próprias referências. Se o arquivo nunca sai da sua máquina, você não precisa empacotar nada.

A outra pessoa não precisa deste add-on

O arquivo abre normalmente sem o Zap Board instalado — sem erros, sem avisos. A pessoa simplesmente vê um Image Editor comum. Se ela editar e devolver, o quadro volta intacto: cards, notas, imagens empacotadas e qual editor era um quadro sobrevivem à ida e volta por um Blender que nunca teve este add-on instalado.

Se algo der errado

Há um botão Report a Bug na barra lateral do quadro e em Edit › Preferences › Add-ons › Zap Board. Ele abre um formulário no seu navegador já preenchido com a versão do add-on, a versão do Blender e o sistema operacional. Nada é enviado pelo Blender em si.
